## Contributing

I welcome contributions to my open-source projects! If you're interested in contributing, please follow these guidelines:

1. **Choose a Project**: Decide which project you'd like to contribute to (infra-foundry or jotai-transaction).

2. **Fork the Repository**: Create your own fork of the project on GitHub.

3. **Create a Branch**: Make a new branch for your feature or bug fix.

4. **Make Changes**: Implement your changes, following the project's coding standards and guidelines.

5. **Write Tests**: Add or update tests as necessary to cover your changes.

6. **Document Your Work**: Update the documentation to reflect your changes, if applicable.

7. **Submit a Pull Request**: Open a pull request from your fork to the main repository. Provide a clear description of your changes and their purpose.

8. **Code Review**: Be open to feedback and be prepared to make adjustments to your code based on the review process.

For more detailed contributing guidelines, please refer to the CONTRIBUTING.md file in each project repository.
